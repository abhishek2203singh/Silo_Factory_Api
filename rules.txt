Master Silo Routes (ms-silo):
- ms-silo:create => Create a new silo in the master table
- ms-silo:update => Update the silo in the master table
- ms-silo:delete => Delete the silo in the master table
- ms-silo:by-id => Get a silo by ID from the master table
- ms-silo:all => Get all silos from the master table
